# getIMUbySerialPort
sanchi IMU is a domestic product.here is the c++ source to get IMU data by SerialPort

how to run?
1. config the serial port Num ,such as COM10,and Baud rate ,for example ,115200.
2. then it will work for 100S of sanchi IMU(the data and information can be found on www.taobao.com)

3. if you use other sensor,serial port also work ,but you should config the transferred data.
